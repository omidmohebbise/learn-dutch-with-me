Welcome to **dutchwithme.nl** - Your companion for learning the Dutch language!

This is a [Next.js](https://nextjs.org/) project designed to help people learn Dutch in an interactive and engaging way.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To build the site for production:

```bash
npm run build
```

## Deploying to GitHub Pages

This site is configured to be deployed to GitHub Pages at **dutchwithme.nl**.

The CNAME file is already configured with your custom domain.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React 18** - UI library
